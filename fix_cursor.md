# Cursor Fix Instructions

## Steps to Fix Cursor File Opening Issue

1. **Reload Cursor Window**
   - Press: Cmd+Shift+P (Mac) or Ctrl+Shift+P (Windows/Linux)
   - Type: "Developer: Reload Window"
   - Press Enter

2. **Clear Cursor Cache**
   - Close Cursor
   - Delete: ~/Library/Application Support/Cursor/Cache
   - Restart Cursor

3. **Try Opening File**
   - Use Cmd+P (Mac) or Ctrl+P (Windows/Linux)
   - Type: SUBMISSION.md
   - Select the file

4. **If Still Not Working**
   - Right-click on SUBMISSION.md in file explorer
   - Select "Open With" -> "Text Editor"
   - Or use terminal: `code SUBMISSION.md`

# Cursor 升级说明

## 📋 如何查看 Cursor 更新日志

### 方法 1：在 Cursor 中查看（推荐）

1. 打开 Cursor
2. 点击菜单栏：**Help** → **Release Notes**（或"更新日志"）
3. 查看最新版本的更新内容

### 方法 2：通过命令面板

1. 按 `Cmd + Shift + P`（Mac）或 `Ctrl + Shift + P`（Windows/Linux）
2. 输入 "Release Notes" 或"更新日志"
3. 选择相应选项

### 方法 3：访问官网

- 访问：https://cursor.ac.cn/changelog
- 查看所有版本的更新日志

## 🚀 最新版本主要更新（2025年）

### 1. Agent 模式全面升级 ⭐

- **默认模式**：Agent 模式现已成为默认模式
- **更强大的 AI 编程体验**：操作更加流畅自然
- **统一体验**：所有 AI 功能都在 Agent 模式下运行

### 2. Composer 与 Chat 结合使用

- **共享上下文**：新的 Chat/Composer 组合界面
- **同一窗口操作**：允许在同一窗口内使用 Agent 和 Chat
- **增强上下文共享**：操作更便捷

### 3. 用户界面（UI）升级

- **界面更清爽**：UI 进行了全面改版
- **更简洁大方**：界面更加简洁，提升了用户体验

### 4. 新增功能

#### Deepseek R1 思维 UI
- 引入了新的思维 UI
- 提升了工作效率

#### MCP 服务器配置
- 新增了 MCP 服务器配置功能
- 方便用户进行个性化设置

### 5. 可靠性和性能改进

- **内存使用优化**：减少了内存使用
- **性能提升**：提升了稳定性
- **键盘快捷键**：所有键盘快捷键现在都可在"键盘快捷键"菜单中找到
- **自定义快捷键**：用户可以自定义或修改快捷键

### 6. 抢先体验选择加入

- **Beta 功能**：用户现在可以从"设置" > "Beta" > "更新频率"中选择加入抢先体验
- **提前体验新功能**：可以提前体验最新的功能

## 🔍 关于聊天面板位置调整

### 重要说明

根据最新版本的更新，**聊天面板的位置调整功能可能已经改变**：

1. **检查命令面板**：
   - 按 `Cmd + Shift + P`
   - 搜索 "panel" 或 "chat" 或 "sidebar"
   - 查看是否有相关命令

2. **检查设置**：
   - 按 `Cmd + ,` 打开设置
   - 搜索 "chat"、"panel"、"sidebar"、"layout"
   - 查看是否有面板位置相关的设置

3. **检查菜单**：
   - 查看 **View** 菜单
   - 查看是否有 "Move Panel" 或类似选项

4. **尝试拖拽**：
   - 如果面板可以拖拽，尝试拖拽到右侧
   - 注意：新版本可能改变了拖拽方式

## ⚙️ 更新设置

### 检查更新频率

1. 打开设置：`Cmd + ,`
2. 搜索 "update" 或"更新"
3. 设置更新频率：
   - **自动更新**：有新版本时自动更新
   - **手动更新**：需要手动检查更新
   - **Beta 版本**：提前体验新功能

### 检查当前版本

1. 打开 Cursor
2. 点击菜单栏：**Cursor** → **About Cursor**
3. 查看版本号

## 🔧 如果遇到问题

### 1. 重置设置

如果升级后出现问题：

```bash
# 备份当前设置
cp ~/Library/Application\ Support/Cursor/User/settings.json ~/Desktop/settings_backup.json

# 清理缓存
rm -rf ~/Library/Application\ Support/Cursor/Cache
rm -rf ~/Library/Application\ Support/Cursor/CachedData
```

### 2. 查看错误日志

1. 打开命令面板：`Cmd + Shift + P`
2. 输入 "Developer: Toggle Developer Tools"
3. 查看 Console 标签页中的错误信息

### 3. 反馈问题

- 访问 Cursor 官方反馈渠道
- 或访问 GitHub Issues（如果有）

# Cursor 聊天面板问题解决方案

## 问题 1: 聊天面板位置调整（从左侧移到右侧）

### 方法 1: 拖拽面板
1. 将鼠标悬停在聊天面板的**标题栏**上（显示 "Chat" 或 "New Chat" 的地方）
2. **按住鼠标左键不放**
3. **拖动面板到右侧边缘**
4. 当看到右侧出现一个蓝色高亮区域时，释放鼠标
5. 面板会停靠在右侧

### 方法 2: 右键菜单
1. 在聊天面板的**标题栏**上**右键点击**
2. 查看是否有以下选项：
   - "Move to Right" / "移动到右侧"
   - "Move to Side" / "移动到侧边"
   - "Dock to Right" / "停靠到右侧"
3. 选择相应选项

### 方法 3: 快捷键
- 按 `Cmd + L` 打开/关闭聊天面板
- 打开后，尝试拖拽到右侧

### 方法 4: 设置中查找
1. 按 `Cmd + ,` 打开设置
2. 搜索 "chat" 或 "panel"
3. 查找 "Chat Panel Position" 或类似选项
4. 设置为 "Right" 或 "右侧"

## 问题 2: 导出聊天记录

### 方法 1: 查找导出按钮（新版本可能位置不同）
1. **检查聊天面板顶部**：
   - 查看是否有三个点（`...`）菜单按钮
   - 查看是否有齿轮图标（设置）
   - 查看是否有下载图标（导出）

2. **检查聊天消息区域**：
   - 右键点击某个消息
   - 查看是否有 "Export" 或 "Copy" 选项

3. **检查命令面板**：
   - 按 `Cmd + Shift + P` 打开命令面板
   - 搜索 "export chat" 或 "导出聊天"
   - 查看是否有相关命令

### 方法 2: 使用脚本提取（如果上述方法无效）

### 方法 3: 手动复制（临时方案）
虽然手动复制很麻烦，但作为临时方案：
1. 在聊天面板中，按 `Cmd + A` 全选
2. 按 `Cmd + C` 复制
3. 粘贴到文本编辑器或 Markdown 文件中保存

### 方法 4: 检查 Cursor 更新日志
1. 打开 Cursor
2. 查看 "Help" → "Release Notes" 或 "更新日志"
3. 查找最新版本中关于导出功能的说明

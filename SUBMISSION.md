# WeChat Article Parser MCP

## Repository

https://github.com/chongsheng007/wechat-article-parser-mcp

## Project Description

A MCP server for parsing WeChat public account articles, built with FastMCP framework.

## Core Tools

1. Article Parser Tool - Parse complete article content
2. Metadata Extraction Tool - Extract title, author, publish time
3. Image Extraction Tool - Extract all image URLs

## Tech Stack

- Framework: FastMCP 2.13.0.2
- Language: Python 3.10+
- Dependencies: BeautifulSoup4, requests, html2text, Pydantic
